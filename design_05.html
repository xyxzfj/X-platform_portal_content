Windows Phone App Studio

微软发布的Windows Phone App Studio为没有编程基础的人提供了一条制作属于自己的app的途径，任何“零基础”的用户都可以通过访问Windows Phone App Studio来免费使用这个工具，其意义在于，通过简单的内容添加和样式选择，实现Windows Phone应用的定制，它不仅屏蔽了Windows Phone应用开发的技术细节，使得没有相关开发经验的普通用户可以创建Windows Phone平台的App，而且可以完成应用原型的快速开发，方便有经验的程序员快速实现其应用程序的原型。

在 Windows Phone App Studio 中，用户可以选择“无模板”和“模板”两种编辑模式，在“模板”编辑模式下，包含了企业、家庭、运动、购物、餐饮、书籍、电影等不同主题的模板，并且在各个模板中已经提供了相关的照片和内容，用户直接更改模板中不符合自己需求的图片和文字即可。而“无模板”和“模板”编辑模式其实十分类似，只是没有了预先设定好图片以及框架内容。

决定使用哪种编辑模式后，即可正式进入操作阶段。操作一共分为四步——填写app信息（标题、描述和logo）、填充app内容、设定app样式、提交。每一步操作均可直接在右侧的模拟屏中预览，实时刷新变更的信息。

第二步填充内容中，用户可以创建不同的页面（section），section 中可以选择的内容形式有 Collection、RSS、YouTube、Html5，选择不同形式后需创建一个数据来源的数据库（DataSource），来存放视频、图文等内容源，而在 SectionPage 中可以选择这些内容的排列样式。

内容填充完成后，即可进入第三部更改整体的app样式。不过这里的更改空间较小，只可以改动 app 的主题颜色，以及背景样式，具体的按钮等 UI 是无法由用户自己改动的。

在确认没有问题之后，可以点击Generate按钮，生成应用。生成应用的速度非常快，一般在几分钟就可以在注册的邮箱中收到邮件，通知应用已经生成，可以点击下载。

在下载的页面，我们可以选择下载其生成的源代码、生成的XAP包，甚至可以通过扫描二维码来直接安装。通过源代码，专业开发者可以进行修改，添加一些该平台无法完成的功能和特性。

地图

高德地图

高德地图Windows 8 API 是一套高效、易用的地图应用开发调用接口（Windows Runtime Component）。使用高德地图Windows 8 API，开发者可以轻松地在Windows 8应用中实现地图和LBS功能。具有以下特性：领先的地图渲染技术，丰富的功能接口，出色的定位性能，以及覆盖全国的海量兴趣点、道路及公交数据。

支持平台：

●  Android。

●  IOS

●  Windows8

●  WP8

SuperMap iClient

SuperMap iClient包含Windows Store apps 和Windows Phone 8 两套开发包，基于Microsoft Windows Runtime和Windows Phone 8技术原生开发；可访问SuperMap iServer Java 6R服务，天地图、BingMaps等第三方服务及OGC标准服务；快速构建全新体验的Windows 8商店风格地图应用和Windows Phone 8地图应用。

支持平台：

●  Windows8

●  WP8

支付

移动快捷支付

快捷支付是支付宝为开发者提供的应用支付专业解决方案，利用系统分享机制，给用户最具win8特色的支付体验，完成付款后自动返回应用，保证了付款过程清晰、安全、便捷、流畅。开发者集成后，即可支持8亿支付宝用户使用余额、100多家银行卡和话费卡等多种方式完成付款。

支持平台：

●  Android

●  IOS

●  Windows8

MX Reader 开发模板

Modern Reader是一套开源C#编写支持Win8和windows phone 8 双平台阅读器模板，拥有良好的交互体验，支持触摸、章节、书签、字体、自动记录进度、昼夜模式等功能，开发人员可以借助该模板快速生成Win8/WP8书籍，目前支持txt格式的电子书导入，使用过程中读者可以通过阅读管理功能在章节、书签、目录间切换，Win8版还提供SnapView的贴靠视图支持。

支持平台：

●  Windows8
